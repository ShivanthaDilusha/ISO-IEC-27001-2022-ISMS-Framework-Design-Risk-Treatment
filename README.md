# ISO-IEC-27001-2022-ISMS-Framework-Design-Risk-Treatment
Designed and developed a comprehensive ISMS core documentation pack for a simulated IT service provider, DSM IT (Pvt) Ltd in strict alignment with the ISO/IEC 27001:2022 standard. This project simulated real-world consulting to transition an organization from ad-hoc security practices to a formalized, audit-ready compliance posture.
